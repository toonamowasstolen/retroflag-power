# Why

> *"Never lose today's excitement in tomorrow's implementation."*

This project began with a simple idea.

Replace an aging Python shutdown script.

That was the original goal.

Along the way, something unexpected happened.

We realized we weren't trying to modernize a script.

We were trying to improve an experience.

The GPi Case 2 is already a wonderful handheld.

It captures the feeling of playing classic games on dedicated hardware.

But every time the power switch is flipped, the illusion is broken just a little.

Booting Linux.

Waiting.

Navigating menus.

Finding the last game.

Loading a save state.

All of those little moments remind the player they're using a computer.

We want to make those moments disappear.

We want the handheld to feel less like Linux and more like a console.

Like picking up a Game Boy.

Like opening a Nintendo Switch.

Like continuing an adventure instead of launching an application.

This project is about respecting the hardware.

Respecting the software that came before us.

Respecting the people who will maintain this project long after we write it.

Most of all, it's about respecting the people who use it.

We believe engineering is both technical and creative.

Architecture matters.

Documentation matters.

Naming matters.

User experience matters.

The code matters.

The feeling matters too.

Every line of code should have a purpose.

Every document should teach.

Every interface should make someone smile.

Every commit should leave the project better than we found it.

We're not trying to write clever software.

We're trying to build thoughtful software.

Software that someone discovers years from now and enjoys reading.

Software that contributors feel welcome to improve.

Software that quietly disappears into the background while the player enjoys their game.

If we succeed, users won't think about GPIO.

They won't think about systemd.

They won't think about RetroArch.

They'll simply flip the power switch...

...and continue their adventure.

That's why this project exists.

Welcome.

Let's build something we're proud of.
